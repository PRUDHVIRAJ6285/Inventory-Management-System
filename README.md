# Inventory Management System

A web-based inventory management system built using Java (Servlets, JSP), MySQL, HTML, CSS, and JavaScript.

## Features
- Admin login authentication
- Product management (Add, Edit, Delete, Search)
- Category management
- Stock tracking
- Low stock alerts
- Dashboard with analytics
- Sales / Orders module
- Automatic stock deduction
- Reports

## Technologies Used
- Java (JSP, Servlets)
- MySQL
- JDBC
- HTML, CSS, JavaScript
- Apache Tomcat

## Architecture
- 3-Tier Architecture (Presentation, Business, Data layer)

## How to Run
1. Import database using `database/schema.sql`
2. Deploy project in Apache Tomcat
3. Access via:
   http://localhost:8080/inventory
